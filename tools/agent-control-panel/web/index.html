Make the stack legible.

This panel turns a local agent setup into something an operator can read, verify, and hand off. It is built to work for Codex, Claude Code, and adjacent local-agent stacks through adapters instead of one-off machine dumps.

Current operating posture

Summarize the stack before you trust it: what the main lanes are, which recovery path exists, and whether the panel snapshot is current.

Execution lanes

The panel assumes a routing model: browser first, desktop when native UI or visible truth matters, dynamic only when normal automation is too static.

Routing principles

These are not implementation details. They are the rules that stop a local automation stack from becoming a black box.
