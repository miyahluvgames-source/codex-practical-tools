# v0.1.0 - 首次公开版

## 版本概览

- 版本：`v0.1.0`
- 状态：建议先作为预发布版本使用
- 适配上游：`siteboon/claudecodeui v1.25.2`

## 本版包含什么

- 手机查看和聊天控制所需的最小覆盖层
- Windows 桌面控制工具源码
- 部署、自检、启动、停止脚本
- 中文优先、面向新手的部署文档
- 设备白名单 / 首次电脑审批这一套安全收敛方案

## 本版不包含什么

- 完整上游源码
- `node_modules`
- 数据库、日志、运行时目录
- 已打包的 `.exe`
- 任何个人环境的真实域名、路径或 secret

## 推荐使用方式

1. 先下载上游 `claudecodeui v1.25.2`
2. 放到 `vendor/claudecodeui-1.25.2`
3. 运行 `scripts/apply-upstream-overrides.ps1`
4. 按 `README.md` 和 `docs/DEPLOYMENT.zh-CN.md` 完成部署

## 发版时特别提醒

- 推荐私网访问，优先 Tailscale
- 不建议把 Node 应用直接暴露到公网
- 新设备第一次登录应经过电脑端批准
- 如果你从旧私有版本迁移，请先轮换旧 secret

## 已知限制

- 主要面向 Windows 单用户自托管场景
- 手机浏览器兼容性通常优于第三方封装 WebView 壳
- 如果你修改了认证或远程暴露方式，需要重新评估安全边界
