# v0.1.0 - First Public Release

## Version overview

- Version: `v0.1.0`
- Status: recommended as a pre-release first
- Supported upstream: `siteboon/claudecodeui v1.25.2`

## What is included

- the minimum override layer required for phone viewing and chat control
- the Windows desktop control tool source
- deployment, self-check, start, and stop scripts
- Chinese-first beginner deployment docs
- the trusted-device and first-time desktop approval hardening model

## What is not included

- full upstream source
- `node_modules`
- databases, logs, or runtime directories
- packaged `.exe` binaries
- any real personal hostname, path, or secret

## Recommended usage

1. download upstream `claudecodeui v1.25.2`
2. place it in `vendor/claudecodeui-1.25.2`
3. run `scripts/apply-upstream-overrides.ps1`
4. follow `README.md` and `docs/DEPLOYMENT.md`

## Important release notes

- private-network access is recommended, ideally through Tailscale
- do not expose the Node app directly to the public internet by default
- require desktop approval for every first-time device login
- if you migrate from an older private build, rotate any old secret first

## Known limitations

- primarily designed for Windows single-user self-hosted usage
- normal mobile browsers are usually more reliable than third-party wrapped WebViews
- if you change auth or exposure behavior, re-evaluate the security boundary
